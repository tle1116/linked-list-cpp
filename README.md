# Linked List Library (C++)

This project implements singly and doubly linked list data structures in C++.

# Features 
- Insert and delete nodes
- Support for singly and doubly linked lists
- Traverse list (foward and backward)

# Concepts Used 
- Pointers
- Dynamic memory allocation
- Object-oriented programming
- Data structures

# Operations 
- push_front
- push_back
- pop_front
- remove

# How to run

Compile:
make
